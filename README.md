# Сапёр

Классический «Сапёр» на ванильном JavaScript. Поле 9×9, 10 мин.

## Как играть

- **Левый клик** — открыть клетку.
- **Правый клик** — поставить/снять флаг 🚩.
- Первый клик всегда безопасен.
- Цифра — сколько мин среди 8 соседних клеток.
- Цель — открыть все клетки без мин. Наступил на мину 💣 — проигрыш.
- 🙂 — новая игра.

## Как запустить

- **Игра** — открыть `index.html` в браузере (сервер не нужен).
- **Тесты** — открыть `tests.html`. Каждый тест даёт строку `OK` или `FAIL`,
  внизу итог (дублируется в консоль, F12).

## Что проверяют тесты

1. `createBoard` создаёт поле нужного размера (9×9 клеток).
2. `createBoard` не расставляет мины до первого клика.
3. `placeMines` расставляет ровно `mineCount` мин.
4. `placeMines` не ставит мины на первую клетку и её соседей.
5. `countAdjacentMines` возвращает точное число мин вокруг клетки.
6. `revealCell` добавляет клетку в `revealed`.
7. `revealCell` — flood fill: пустая клетка открывает все связанные пустые
   клетки и их соседей, не задевая мину.
8. `toggleFlag` ставит флаг на закрытую клетку и снимает повторным кликом.
9. `checkWin` возвращает `false`, пока есть закрытые безопасные клетки.
10. `checkWin` возвращает `true`, когда открыты все безопасные клетки.
11. `createBoard` ограничивает `mineCount` числом мин, которое реально влезет.
12. `placeMines` ставит ровно `mineCount` мин даже при клике в центр.
13. `placeMines` на 3×3 не виснет (поле слишком мало, мин 0).
